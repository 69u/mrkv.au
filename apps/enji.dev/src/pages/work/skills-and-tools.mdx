## Full-Stack Developer

I love creating something that is clean and minimalistic, attractive and has value, and of course, easy to use.

I really enjoy creating scripts with **rich UI components**, including:

- **Web applications**,
- **Documentation pages**,
- **CMS contents layout**,
- **Dashboard layout**,
- and others.

But I still like to make simple website pages such as **landing pages**.
<br /><br />
So, what tools did I feel comfortable with using during this website creation?

### JavaScript

First of all, the programming language that I'm most knowledagble with using is **JavaScript**, although website creation can be done using PHP, Python, etc.

JavaScript has accompanied me for **about {new Date().getFullYear() - 2018} years**.

### Vue JS

I really enjoy building single page application websites and Vue is my go-to library.

I've been using it **since 2020 ({new Date().getFullYear() - 2020} years)** and have become quite proficient with it.

### Tailwind CSS

For now, I love using Tailwind CSS for styling, even though I've been using SCSS for a longer time ({2020 - 2016} years).

I'm still new to Tailwind CSS, but **I'm really comfortable** using it and I think it's a great tool.

### Vueuse Motion

For animations, I am more comfortable using Vueuse Motion. Creating animations using JavaScript has **never been as easy as using Vueuse Motion**.

### Nuxt.js

Since discovering Nuxt.js, I don't think I would want to use any other web framework. **It's just that good!**
